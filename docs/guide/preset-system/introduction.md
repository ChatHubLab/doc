# 预设系统

接下来我们将介绍如何使用预设系统，用于调教模型的对话风格，行为等。

预设系统类似 ChatGPT 的 [custom-instructions](https://openai.com/blog/custom-instructions-for-chatgpt)，不同的是我们的预设系统可以伪造多条回复，内嵌变量，使得更适合角色扮演。不过两者都可自定义模型的对话风格，行为等。
